[전체]
    세로형 탭일 경우 : tablist 요소에 aria-orientation="vertical"
    http://nuli.navercorp.com/sharing/blog/post/1132879

[키보드] : c-tab__anchor
    space, enter: 탭패널 활성화
    방향키 : 탭 버튼 요소 간 이동 (처음인 경우 마지막으로, 마지막인 경우 처음으로 이동 가능)
    home : 첫번째 버튼으로 이동
    end : 마지막 버튼으로 이동

[활성화]
    button : c-tab__anchor
        aria-selected : 활성화 true, 비활성화 false
        tabindex : 활성화 없음, 비활성화 -1
    panel : c-tab__panel
        aria-hidden : 활성화 false, 비활성화 true
