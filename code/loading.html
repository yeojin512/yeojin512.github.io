예시

<div id="★loading-start" aria-live="assertive">
    <span class="sr-only">로딩이 시작되었습니다.</span>
</div>
<div id="★loading-end" aria-live="assertive">
    <span class="sr-only">로딩이 종료되었습니다.</span>
</div>

사용

로딩이 시작되면 <div id="★loading-start" aria-live="assertive"></div>에 role="alert" 속성이 삽입되고, 로딩 내용이 삽입된다.
로딩이 종료되면 <div id="★loading-end" aria-live="assertive">에 로딩 내용이 삽입된다.
각 일정시간(ex,3초) 이후에는 해당 내용 모두 삭제
