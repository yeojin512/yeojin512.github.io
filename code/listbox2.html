사용

[키보드] : listbox__anchor
    enter, space, ALT + 방향키 : 리스트 열림/닫힘
    방향키 : 위아래로 이동, 닫혔을 경우에도 이동
    home : 첫번째 옵션으로 이동
    end : 마지막 옵션으로 이동
    page up, page down : 옵션 3개씩 이동, 닫힌상태에서도 옵션 3개씩 이동
    esc : 리스트 닫히고 버튼으로 포커스 이동
    리스트 열렸을때 현재 활성화된 option에 포커스
[활성화]
    button : listbox__anchor
        활성/비활성화에 따라 열기/닫기 텍스트 변경
    list : listbox__panel
        버튼 클릭시 list로 포커스 이동 및 방향키 포커스 갇힘
        li의 선택된 옵션 id값이 ul의 aria-activedescendant에 반영
        li 선택된 곳에 aria-selected="true" 생성, 이전 삭제

비교

[센스리더기]
    aria-haspopup="listbox" : X
    role="listbox" : 스크립트 작업 후 확인 필요
    role="option" : 스크립트 작업 후 확인 필요
    aria-labelledby : O
    aria-expanded : X
    aria-activedescendant : X
    aria-label : X
    aria-selected : ? (첫번째 일때는 안읽히는데 두번째 인 경우 첫번쨰가 선택안됨이라고 알려줌)
