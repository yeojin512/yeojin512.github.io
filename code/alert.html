modal이랑 디자인이 다를 경우 c-dialog 로 사용, 마크업에서 클래스 변경

사용

[키보드]
    esc : 모달 닫힘
    모달에 첫 진입시에만 class="modal"에 포커스 생성
    모달안에 포커스 가둬야함
[활성화]
    html 태그에 class="has-modal" 클래스 생성

비교

[센스리더기]
    role="alertdialog" : X
    aria-labelledby : O (role를 dialog로 변경할 경우)
    aria-describedby : X
[NVDA]
    role="alertdialog" : X
    aria-labelledby : O (role를 dialog로 변경할 경우)
    aria-describedby : O (role를 dialog로 변경할 경우)
