사용

[키보드] : toggle__anchor
    enter, space : 해당 버튼의 패널 활성화
[활성화]
    button : toggle__anchor
        aria-expanded : 활성화 true, 비활성화 false
    panel : toggle__panel
        aria-hidden : 활성화 false, 비활성화 true

비교

[센스리더기]
    aria-controls : X
    aria-expanded : X
    aria-labelledby : X
    aria-hidden : O
[NVDA]
    aria-controls : X (연결되어있는지 panel에서 안읽어줌)
    aria-expanded : O
    aria-labelledby : X (연결되어있는지 panel에서 안읽어줌)
    aria-hidden : O
