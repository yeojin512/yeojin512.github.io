사용

[기타]
    panel에 tabindex="0" 는 디자인, 기획서 확인 후 제거, 스크롤이 아닌 경우 제거
[활성화]
    button : toggle__anchor
        활성/비활성화에 따라 열기/닫기 텍스트 변경
    panel : toggle__panel
        aria-hidden : 활성화 false, 비활성화 true

비교

[센스리더기]
    aria-controls : X
    aria-expanded : X
    aria-labelledby : X
    aria-hidden : O
[NVDA]
    aria-controls : X (연결되어있는지 panel에서 안읽어줌)
    aria-expanded : O
    aria-labelledby : X (연결되어있는지 panel에서 안읽어줌)
    aria-hidden : O
